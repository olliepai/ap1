Changing a StyleSheet Without Reloading

The goal of this challenge is to have the page load different style sheets based on button clicks. The style sheet is loaded in the <head> tag, specifically inside of the <link> tag.

Add onclick attributes to the buttons below so that they change the stylesheet associated with the page. You will also need to create two different stylesheet files (e.g. blah.css and foo.css) to be used with this challenge; you will most likely want to put these files in the same directory as this file.

Note: you don't need to create light and dark themes; you can choose any two themes that you want. If you want to do cat and dog themes, go for it.
